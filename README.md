## To run locally

```
git clone https://github.com/434huzaifa/AssetTrackr-backend
```
```
cd AssetTrackr-backend
```
```
pip install -r requirements. txt
```
```
python manage.py runserver
```
or if you python >= 12
```
py mange.py runserver
```
